If the contacts app does not automatically open, you may have to download the file here. Different browsers handle downloads in their own unique ways. For example, Safari users on iPhone often find that the VCF file opens directly in the Contacts app, making it easy to add to their contact list.

However, if you're using another browser like Firefox or Chrome, the file may download to your device instead. In this case, please locate the downloaded VCF file in your device's download folder or download history. Once found, you can tap on the file to open and import your new contact information into your Contacts app.
